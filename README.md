## Структура проєкту
Структура цього проєкту побудована на основі `Public API`. Це спосіб реімпортування даних з папок через `index.ts` файлів. Це дозволяє контролювати що потрібно, а що не потрібно імпортувати.
## Стиль написання коду
Наш стиль коду досить простий, але легко читається.

- 4 пробіли (табуляції) для кожного рівня вкладеності для кращої читабельності
- Усі імпорти розміщуємо на самому початку файлу, щоб завжди можна було знайти, звідки все імпортовано
- Назви файлів:
    - файли стилів називаємо: `назва-компоненту.module.css`
    - Файли з компонентами називаємо з великої літери та `camel case`. `Компонент.tsx`
    - Файли з кастомними хуками називаємо через тире. `назва-хука.tsx`
